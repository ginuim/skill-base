# 写作规范详细参考

## 结构模板

### 标准传记（3000-5000字）

```
引言（200-300字）
    └── 用一个具体细节或成就引入，设置悬念

生平背景（400-500字）
    └── 出生、家庭、成长环境（简洁，不铺陈）

核心成就（1500-2000字）
    └── 按时间线或主题叙述
    └── 必须有具体数据和案例
    └── 与同时期其他文明的对比

历史影响（500-600字）
    └── 对中国的影响
    └── 对世界/人类的影响
    └── 后世评价（引用国际观点）

争议与局限（300-400字）
    └── 客观呈现历史争议
    └── 不回避局限性和批评

结语（200字）
    └── 开放式结尾，留待读者评价
```

### 短篇特写（1500-2000字）

适用于人物的一个侧面或一个成就：

```
切入细节（100字）
    └── 一个具体场景或画面

核心论述（1000字）
    └── 围绕一个主题展开
    └── 数据与案例支撑

延伸思考（300字）
    └── 人物在更大历史图景中的位置

开放式结尾（100字）
    └── 引发读者思考
```

## 人物类型写作要点

### 古代思想家

- 思想核心是什么？（用一两句话概括）
- 这个思想在当时的独特性？
- 对后世中国的影响？
- 与同时期古希腊/古印度/古罗马的对比？

### 科学家与发明家

- 具体成就的技术细节（通俗化表达）
- 这个成就在当时的国际地位？
- 知识如何传承？为何失传？
- 对人类文明的贡献？

### 帝王将相

- 重大决策的背景是什么？
- 决策的后果有哪些？
- 同期其他国家的类似人物怎么做？
- 如何被后世评价？为何有这种评价？

### 近现代人物

- 时代背景（战争、革命、建设等）
- 个人选择与时代洪流的关系
- 国际同行的评价
- 对今天的影响

## 事实核查清单

每篇文章完成后逐项确认：

- [ ] 具体年份和年龄
- [ ] 代表作/成就的具体内容
- [ ] 同期国际对比（如有）
- [ ] 国际学术界评价（引用权威来源）
- [ ] 人物行为的动机分析（逻辑链条完整）

## 配图类型参考

| 内容 | 推荐类型 |
|:---|:---|
| 人物肖像 | 历史画像/照片 |
| 事件场景 | 历史画作/复原图 |
| 作品展示 | 实物照片/摹本 |
| 地理分布 | 地图标注 |
| 时间线 | 生平大事记 |

配图占位格式：
```markdown
> [配图建议：描述内容、来源建议、配图逻辑]
```
