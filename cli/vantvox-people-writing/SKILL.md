# VANTVOX 人物栏目写作

> 基于 SOP 项目核心定位：理性观察、文明翻译、逻辑推演。
> 本技能与 `vantvox-blog-writing` 互补——通用编辑标准（敏感词替换、视觉约束等）仍遵循该技能。

## 核心原则

**基本立场：**

- **客观中立**：事实优先，让读者自行评价
- **平视视角**：不神化、不矮化，还原真实的人
- **逻辑驱动**：用因果链条解释人物命运，而非情绪化判断
- **跨文化坐标**：为读者提供同时期其他文明的参照

**叙事底线（硬性禁止）：**

- 口号化表达（"伟大"、"厉害"、"必胜"等）
- 没有数据支撑的空洞评价
- 为尊者讳，回避人物的局限与争议
- 简单二元对立（好人/坏人，英雄/恶棍）

## 三维定位法

写任何人物前，先回答三个问题：

| 维度 | 问题 | 示例 |
|:---|:---|:---|
| **时间坐标** | 他/她生在什么时代？ | 祖冲之生于南北朝（429年） |
| **空间坐标** | 他/她站在什么位置上？ | 江南士族，县学教谕 |
| **问题坐标** | 他/她面对什么困境？ | 五次科举不第，仕途无门 |

**复杂性原则**：真实的人都是复杂的。

错误："他是中国古代最伟大的科学家之一。"

正确："他在圆周率计算上达到了当时世界的顶尖水平，但他更重要的身份是一个'失败者'——五次科举不第，官职微末。他的伟大，恰恰在于：在一个不重视技术的时代，他选择了技术。"

## 写作工作流

### Phase 1: 定位与结构

1. 完成三维定位（时间、空间、问题）。
2. 确定文章类型——标准传记（3000-5000字）或短篇特写（1500-2000字）。
3. 产出详细大纲，标注"跨文化锚点"。
4. 确定文章的**视觉焦点**——需要隐喻配图的核心论点。

### Phase 2: 写作

1. 按结构模板写作（详见 [writing-standards.md](writing-standards.md)）。
2. 遵循 `vantvox-blog-writing` 中的**敏感词替换表**和**叙事框架规则**。
3. 60/30/10 比例：60% 理性分析，30% 人文叙事，10% 诗性反思。
4. 在最抽象或最有情感张力的段落插入视觉锚点标记：`[Visual Anchor: 描述核心逻辑]`。
5. 所有成就描述必须有具体数据支撑，所有对比必须有参照物。

### Phase 3: 质量审核

逐项检查：

- [ ] 所有数据有据可查（具体年份、具体数字）
- [ ] 代表作/成就有具体内容描述
- [ ] 包含同期国际对比（如有可比项）
- [ ] 引用权威来源的国际学术评价
- [ ] 人物行为的动机-因果链条完整
- [ ] 无口号化表达，无空洞评价
- [ ] 呈现了人物的多个面向（成就 + 局限 + 争议）
- [ ] 避免了"事后诸葛亮"的上帝视角
- [ ] 敏感词替换表已应用
- [ ] 配图提示词符合 Bio 栏目视觉原型（双重曝光、深沉肖像、象征物件）

### Phase 4: 交付

1. 产出 `.md` 文件，包含 frontmatter（title, slug, category: Bio, summary）。
2. 文中标注配图位置与对应 Nano Banana 提示词。
3. Slug 简洁有意境（如 `lost-algorithm-zu-chongzhi`）。

## 数据表达规范

**成就描述：**
错误："他在数学上取得了巨大成就。"
正确："他将圆周率精确到小数点后第七位，这个纪录保持了800年。"

**国际对比：**
错误："中国在这方面领先世界。"
正确："在17世纪初，中国在冶金技术上领先欧洲约200年。"

**引用格式：**
"英国科学史学家李约瑟评价说：'……'"

## 语言风格

| 应该 | 不应该 |
|:---|:---|
| 具体、精确 | 空洞、夸张 |
| 理性分析 | 情绪渲染 |
| 逻辑推演 | 结论先行 |
| 多角度呈现 | 单向度叙事 |

## 人物类型速查

不同类型人物的写作侧重点不同，详见 [writing-standards.md](writing-standards.md)。

| 类型 | 核心关注点 |
|:---|:---|
| 古代思想家 | 思想核心、独特性、与古希腊/古印度/古罗马对比 |
| 科学家与发明家 | 技术细节通俗化、国际地位、知识传承与失传 |
| 帝王将相 | 决策背景与后果、同期他国类似人物对比 |
| 近现代人物 | 个人选择与时代洪流、国际同行评价 |

## 配图规范

Bio 栏目视觉原型为**深沉（Depth）**：双重曝光、深沉肖像剪影、象征性物件（书籍、怀表）。

配图占位格式：
```markdown
> [配图建议：描述内容、来源建议、配图逻辑]
```

Nano Banana 提示词关键词：`double exposure portrait`, `deep silhouette`, `symbolic objects`, `vintage atmosphere`, `contemplative mood`

## 参考来源优先级

1. 一手史料
2. 权威学术著作（李约瑟《中国科学技术史》等）
3. 国际学术机构研究报告（哈佛、牛津、斯坦福）
4. 国际主流媒体（《纽约时报》、《经济学人》、BBC）
5. 学术论文数据库（JSTOR、知网）

引用时注明出处和年代，避免过度依赖单一来源。
